### 🛠️ Setup Instructions

1. **Install Dependencies**
   ```bash
   cd ijl-roi-calculator-fixed
   npm install
   # or
   pnpm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   # or
   pnpm run dev
   ```

3. **Build for Production**
   ```bash
   npm run build
   # or
   pnpm run build
   ```
